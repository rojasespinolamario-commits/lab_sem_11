## Descripción
Para esta tarea hemos hecho uso del patron de diseño Decorator para representar las bebidas de la cafeteria y sus complementos
Las bebidas pueden personalizarse agregando:

- Leche
- Moca
- Soya
- Crema

sin modificar las clases originales de cada bebida.

### Componentes principales

- Cafe (componente abstracto)
- Expreso
- Descafeinado
- Batido
- Tostado Negro

### Decoradores

- Leche
- Moca
- Soya
- Crema

Cada decorador agrega una descripción y un costo adicional a la bebida.

## Nueva funcionalidad

Se agregaron tres tamaños para las bebidas:

- Normal (N)
- Mediano (M)
- Grande (G)

Tanto las bebidas como los complementos tienen precios distintos según el tamaño seleccionado.

## Uso de HashMap

Se utilizó HashMap para almacenar los precios de bebidas y complementos según el tamaño.
